# Day2day Productivity

A calm, responsive productivity dashboard for organizing daily tasks, tracking focus, and reviewing progress.

## Features

- Responsive desktop and mobile layout
- Dynamic greeting based on local time of day
- User name prompt with saved profile name
- Sidebar navigation for switching views
- Mobile bottom navigation
- Task sections for:
  - Dashboard
  - Schedule
  - Tasks
  - Analytics
  - Focus Mode
  - Journal
  - Settings
- Add-task input with task type selection
- Local storage persistence for saved data
- Personalized reminder and reflection text

## Tech Stack

- HTML
- CSS
- JavaScript
- LocalStorage

## How to Use

1. Open the app in your browser.
2. Enter your name when prompted.
3. Use the sidebar on desktop or the mobile navigation on smaller screens.
4. Add tasks and explore each section to update your productivity space.

## Mobile Experience

On mobile, the layout adapts to a simpler view with navigation controls at the bottom so you can switch sections easily on the go.

## Data Storage

The app uses browser local storage to keep your tasks and profile name saved between visits.

## Folder Structure

- `index.html` — page structure
- `style.css` — styling and responsive layout
- `script.js` — app logic, navigation, greetings, and storage

## Notes

This project is designed as a lightweight front-end productivity dashboard and does not require a backend.
